# ProyectoTicmasCVSegundaEntrega
Trabajo final del curso Front-end Acadamy Ticmas, Argentina Programa 4.0.
Mi proyecto final integrador de Front end Academy - 2022 de Argentina programa 4.0

Mi consigna era crear un cv online que me permitiese llevar a la práctica lo trabajado en el curso. Utilicé la página “RandomUserGenerator” para obtener la información del postulante. 
Comencé mi proyecto creando un archivo “html.index” donde plasmé esa información dividiéndola primeramente en un “header” y luego en secciones. Utilizando CSS pude obtener diversos estilos y con JavaScript logré desarrollar un botón desplegable donde se muestran las habilidades.
